---
description: How to push to GitHub and deploy to Vercel/Netlify
---

1. Initialize git and commit: `git init && git add . && git commit -m "initial commit"`
2. Create a repo on GitHub and follow the instructions to push:
   `git remote add origin <URL>`
   `git branch -M main`
   `git push -u origin main`
3. Connect the repo to Vercel or Netlify for automatic deployment.
